## ✨ Features

- 🎨 **Smooth Animated Navbar** — scroll-aware with elegant transitions
- ⌨️ **Special Typing Animation** — dynamic role text in hero section
- 🌈 **Colorful 3D Objects** — interactive Three.js / Framer Motion hero elements
- 👤 **Clean About Section** — personal intro with highlights
- 📊 **Animated Skills Section** — progress bars with tech stack icons
- 💼 **Projects Section** — featured projects with tags, links & live preview
- 📬 **Contact Section** — full order/inquiry form with service, budget & timeline
- 🦶 **Footer** — social links, quick nav, branding
- 🔗 **`/projects` Route** — dedicated page to see all projects

## ⚙️ Environment Variables

Create a `.env` file in the root:

```dotenv
VITE_API_URL=https://your-backend.onrender.com
```

---

## 🏃 Run Locally

```bash
# Clone the repo
git clone https://github.com/mashhudur/portfolio.git

# Install dependencies
cd portfolio
npm install

# Add your .env file
# Then start
npm run dev
```

---

## 📦 Build for Production

```bash
npm run build
```
